# snap-parser
`snap-parser` is a highly-experimental tool for parsing the XML-based project format used by the Snap! family of programming languages.

The Snap! family includes but is not limited to:
- Snap!
- Split!
- Snavanced!
- Jameson

For documentation, please see the wiki.
